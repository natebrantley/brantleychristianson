# Brantley Christianson Real Estate

Next.js site for Pacific Northwest real estate (Oregon & Washington markets).

## Development

- **Node:** Use the version in [.nvmrc](.nvmrc) (e.g. `nvm use`).
- **Scripts:** `npm run dev` | `npm run build` | `npm run typecheck` | `npm run lint` | `npm run test:e2e`
- **CI & Cursor:** This repo uses [Cursor Team Kit](docs/TEAM_KIT.md) for CI, PR review, and shipping. See [docs/TEAM_KIT.md](docs/TEAM_KIT.md) for how to use each skill.
